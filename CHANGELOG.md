## 0.1.0 - 2026-04-13

First public open-source release.

### Added

- Native macOS file browser built with Swift and AppKit
- Sidebar favorites and mounted-volume browsing
- List, column, icon, and gallery views
- Preview pane and dual-pane browsing
- Breadcrumb navigation and Go to Folder
- Search and filter support for the current browsing workflow
- Copy, move, duplicate, rename, batch rename, compress, and extract actions
- Quick Look, drag and drop, and context menus
- Window restoration and general workspace persistence
- MIT licensing, contribution guidelines, and a local install script

### Changed

- README rewritten around public usage and local source builds
- Repository prepared for public contributions and pull-request-based development
